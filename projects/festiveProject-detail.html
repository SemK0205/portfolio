문제 해결 경험

01

실시간 알림 시스템 구현

실시간 통신

문제: 관리자가 사용자의 문의, 부스 신청, 신고 등을 실시간으로 파악하지 못해 응답 지연이 발생했습니다.

해결: WebSocket을 활용한 실시간 양방향 통신을 구현했습니다. 사용자의 액션이 발생하면 즉시 관리자에게 알림이 전달됩니다.

결과: 평균 응답 시간 70% 단축, 빠른 처리로 인한 사용자 만족도 크게 향상
02

위치 기반 축제 정보 제공

외부 API 연동

문제: 사용자 위치를 기반으로 한 맞춤형 축제 정보와 날씨 정보를 제공하는데 어려움이 있었습니다.

해결: 브라우저의 Geolocation API와 외부 날씨 API를 연동하여 사용자의 현재 위치를 파악하고, 거리 계산 알고리즘을 통해 가까운 축제 순으로 정렬하는 시스템을 구축했습니다.

결과: 개인화된 축제 정보 제공으로 사용자 편의성 및 정보제공의 유용성 증가
03

Toast UI 에디터 통합

사용자 경험

문제: 기본 텍스트 에리어로는 사용자들이 풍부한 콘텐츠를 작성하기 어려웠고, 게시글의 가독성이 떨어졌습니다.

해결: Toast UI 에디터를 도입하여 마크다운 지원, 이미지 삽입, 테이블 작성 등 다양한 기능을 제공했습니다. 또한 에디터와 백엔드 간의 데이터 파싱 로직을 구현했습니다.

결과: 콘텐츠 품질 및 사용자 편의성 증가
